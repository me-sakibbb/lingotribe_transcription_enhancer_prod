Lingotribe Transcription Enhancer - Settings

Quick Start Guide

Keyboard Shortcuts:

  • Alt+R - Apply bulk replacements to selected text
  • Alt+F - Format selected text (or entire field)
  • Alt+N - Convert word numbers to digits
  • Alt+A - Apply ALL (Alt+R + Alt+F + Alt+N combined)
  • Alt+P - Show/hide floating word picker
  • Enter or Tab - Accept autocomplete suggestion

💡 Tip: Configure your preferences below and click "Save Settings" at the bottom.

General

Control whether the extension is active. When disabled, all features (autocomplete, replacements, formatting) will be turned off.

Autocomplete Shortcuts

How it works: As you type, matching shortcuts will appear in a dropdown menu. Press Enter or Tab to insert the full text.

Examples:
• Type "brb" → Suggests "be right back"
• Type "omw" → Suggests "on my way"
• Type "thx" → Suggests "thank you"
• Type "addr" → Suggests your full address

Word Replacements

How it works: When you type a word and press Space, it will automatically be replaced with the correct spelling. Also if you press Alt+R, it will replace all the words in the current input field. Perfect for fixing common typos!

Examples:
• Type "teh " → Automatically becomes "the "
• Type "cant " → Automatically becomes "can't "
• Type "dont " → Automatically becomes "don't "
• Type "recieve " → Automatically becomes "receive "

Floating Word Picker Items

How it works: Press Alt+P to show/hide a floating window with quick-access words or phrases. Click any item to insert it at your cursor.

Use cases:
• Frequently used phrases: "Thank you for your time", "Best regards"
• Technical terms: "JavaScript", "Python", "SQL"
• Emoji or special characters: "✓", "✗", "→"
• Common responses: "Yes, I agree", "No, thank you"

Formatting Rules (Alt+F)

How it works: Select text and press Alt+F to apply all enabled formatting rules. If no text is selected, it formats the entire input field.

What gets formatted:
1. Removes unwanted words (see list below)
2. Cleans text inside brackets (lowercase, removes punctuation)
3. Adds single space after punctuation
4. Removes double spaces
5. Capitalizes first letter of sentences

These exact phrases will be completely removed when you press Alt+F. Useful for removing filler words from transcriptions.

Common examples:
• Audio cues: [inhale], [exhale], [cough], [pause]
• Filler words: [um], [uh], [hmm], [like]
• Editor notes: [music], [laughter], [background noise]

Before: "So [um] I think [inhale] we should proceed."
After: "So I think we should proceed."

Number Conversion (Alt+N)

How it works: Press Alt+N to convert ALL number words to digits throughout the text.

Examples:
• Input: "I have one apple, two oranges, and three bananas"
• Output: "I have 1 apple, 2 oranges, and 3 bananas"

• Input: "Chapter one discusses ten key points"
• Output: "Chapter 1 discusses 10 key points"

• Input: "twenty-five" (with hyphen)
• Output: "20-5" (converts each word separately)

Supported: zero through nineteen, twenty, thirty, forty, fifty, sixty, seventy, eighty, ninety, hundred, thousand

Apply All Transformations (Alt+A)

How it works: Press Alt+A to apply ALL transformations at once in a single operation. This is the most powerful shortcut that combines everything!
